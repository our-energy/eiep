# Electricity Information Exchange Protocol (EIEP)

A PHP library for working with the Electricity Authority's EIEP data files. Supports PHP 7.1+.

Currently supported protocols and versions;

| Protocol  | Version(s) |
| ------------- | ------------- |
| EIEP3  | 10.0  |
| EIEP13a  | 1.1  |
